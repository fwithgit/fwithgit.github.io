Eclipse运行一个Servlet项目

发表于2024-04-02|更新于2024-04-03

环境准备

要用Eclipse 的企业版 Eclipse EE 新建Web项目，Eclipse EE下载链接Eclipse EE，其实Idea什么的也可以，就是挺麻烦的，Eclipse相对简单一点。点击DownLoad下载

jdk我是1.8.0_381的，tomcat是9.0.27，如果你是在学校机房弄的话，基本上都是配的刚刚好的，不用你怎么操作，我这是在自己电脑弄得，小麻烦。

你要搭建Servlet的话要设置好Servlet Spec, Tomcat, jdk版本能对应上，在Tomcat那里有对应的关系，Apache Tomcat® - Which Version Do I Want?

比如我的jdk版本是1.8的，那我只能选Tomcat9.0.x以上的，你是11的只能选10.1.x以上的，至于Servlet Spec这个待会才用到，这个不用下载东西。

项目创建

打开Eclipse ee，点击左上角的File，New，Other

选择Web下的Dynamic Web Project

填写Project Name，没有什么特别的要求，先不要点Next或Finish，这里有关选项是Dynamic web module version，这里要和刚刚Tomcat那里的要求对应上，比如我的jdk是1.8，Tomcat是9.0.x的，我应该选4.0，选好Finish就可以了

OK，现在项目建好了。

有一点要注意，Servlet4.0以上（不包含4.0）由Eclipse来维护，导入的包名是不一样的。我这个是4.0的，如果你是5.0及以上的，要按照这段话进行更改Servlet 5.0：主要是把javax.servlet包名改成了jakarta.servlet ；

我们先找到tomcat在哪，tomcat所在的文件夹的lib里面有我们需要的包，在Eclipse EE里面导入它
右击项目的Properties，
点击Java Build Path，Libraries，Add External JARs

找到刚刚Tomcat路径那个Servlet的jar包点击打开，点击Apply and close
此时报错都消失了，点击Eclipse的Window\Preferences
选择Server\Runtime Environment\Add

点击Apache选择对应的Tomcat版本
选择Browse你的Tomcat路径，然后点Finish。题外话，下面这个Download and install这个功能好像是可以帮你找到适合的tomcat版本，但是我试了一下不行…
回到这里后点击下面的Servers和它下面的一串蓝色英文

继续选择对应的Tomcat然后点next
这里我刚刚弄过一次，理论在Available上你那里就只有一个项目，将它add到Configured那里去，如果它已经在Configured了就不管他了，点击Finish
此时我们选择项目的Run AS，Run on Server
点一下那个服务器，然后Finish就可以了
等它一会会自己在浏览器弹出来，或者你自己访问http://localhost:8080/你的项目名/TestServlet01 也可以，自己弹出来的可能会报404，你要在路径后面再加上参数TestServlet01
