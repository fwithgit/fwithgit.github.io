Spring Boot快速入门 Java web笔记

发表于2024-03-01|更新于2024-03-09

创建Spring boot工程

  在idea中对项目新建一个Module，Generators选择Spring Initializr，好像是因为新的springboot项目不支持老版本的jdk了，我们要在Server URL中更换为阿里云的网站 https://start.aliyun.com 才能使用jdk11创建spring boot项目，当然你也可以使用jdk17或者jdk21来直接创建，在右侧你还会看到以下配置项

配置项 作用
Server URL 选择从哪个服务器下载新建spring代码，https://start.aliyun.com
Name 项目名称（先不用写）
Location 存储位置
Language 编程语言 Java
Type 类型 Maven
Group 组织名（反写域名） com.itheima
Artifact 模块名 springboot-web-quickstrart
Package name 包名 com.itheima
JDK JDK版本，我选11
Java JAVA版本，我选11
Packaging 应该是包的格式，选JAR

  设置完上面的内容会到新的界面，选择spring boot版本（我是2.6.13）选择Web然后选择Spring Web，create然后等待下载。

  如果你也是阿里云的镜像来创建的spring boot web项目，你会发现他的文件会和黑马程序员老师的文件有所差异，不过少的都是用不上的。但是他还会多一个demo的文件，接下来你跟着黑马程序员的老师敲一些案例时会冲突，请先删掉demo的文件夹。
