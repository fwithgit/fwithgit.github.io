2024上半年软件设计师备考笔记

发表于2024-03-29|更新于2024-03-29

下午题介绍_哔哩哔哩_bilibili这个UPzst_2001的个人空间 (bilibili.com)讲的挺好的，五星好评👍
软考分为上午题和下午题，可以去看看 软件设计师（第五版） 这本书，官方指导

考试内容介绍

上午题

上午题有75道选择题，共75分，45分算合格。
常见考点：

  • 进制转换，原码、反码、补码、移码这种
  • 浮点数
  • 校验码
  • CPU组件
  • 计算机可靠性
  • 计算机安全性，加密，公钥私钥这种

下午题

下午题有六道题，共75分，其中一到四的题必答（共60分），五六选择其中一道作答（共15分），五一般是C++实现，六是Java实现。

题目 题目内容 所属知识点 难度
一 DFD数据流图 结构化开发方法 简单
二 E-R图 数据库 简单
三 UML图（类图+用例图） 软件工程 简单
五 C++ 23种设计模式代码填空 有难度
六 Java 和C++类似 有难度
四 C语言 算法分析与设计 有高有低

这里面的题还会细分比较多的小题，概念题等。第五六题会有面向对象什么的，第四题会问复杂度，给实例求解这种。
